# Dark / Light Toggle

A simple front-end project for practicing theme toggling between Dark and Light modes.  
Built using **HTML5**, **CSS3**, and **JavaScript**, focusing on layout, styling, and basic interactivity with icons.  
This project is for educational purposes and portfolio demonstration only.

## Features
- Two side-by-side buttons for Dark  and Light  modes
- Hover animations with smooth scaling and color change
- Background color changes on click
- Persistent theme state using `localStorage`
- Responsive layout and modern design
- Educational project for practice purposes
